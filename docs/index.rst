Welcome to python-packer's documentation!
=========================================

Packer is a tool written by Hashicorp for creating images in multiple Hypervisors/IaaS providers. The tool is written in Go and provides a CLI for executing its commands.

python-packer aims to provide a Pythonic interface for running `Packer <http://www.packer.io>`_.
